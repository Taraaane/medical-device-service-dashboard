# Medical Device Service Dashboard

This project is a comprehensive analytical dashboard designed to manage and monitor medical device service operations. Its primary goal is to provide a clear overview of maintenance costs, system downtime, and equipment performance across different geographic regions.

## 📊 Key Features

This dashboard includes the following analytical sections:
* **KPIs Overview**: Displays high-level metrics including "Total Maintenance Cost" and "Average System Downtime".
* **Failure Analysis**: Analyzes total failure events categorized by device type (e.g., X-Ray Machine, Defibrillator, MRI Scanner, etc.).
* **Cost Distribution**: Visualizes the distribution of costs broken down by region.
* **Age-Cost Correlation**: Provides a trend analysis of "Average Maintenance Cost by Device Age" to help identify cost patterns.
* **Interactive Filtering**: Features a country-based slicer to allow users to filter data and examine regional performance specifically.

## 🛠 Tech Stack
* **Power BI**: The primary tool used for visual design, data modeling, and performance monitoring.

## 🚀 How to View
This project was designed in Power BI Desktop. To view the source file, you can open the `.pbip` or `.pbix` file using Power BI Desktop.
